This app is a box office movies viewer for Android.  It pulls up to date data from Rotten Tomatoes using their API.  The movie data is pulled in the JSON format and decoded.  This app was created at the codepath intro to Android workshop.

Functionalities:
 - pull up to date box office movie data
 - display movie title, Rotten Tomatoes score, and staring actors
 - scroll through current movies
